# 吐槽伴侣（Rant Companion）

一个用于 SillyTavern 的悬浮吐槽插件。  
你可以一边跑剧情，一边让 AI 以聊天风格吐槽当前内容，并将吐槽历史按角色分开保存。

## 功能亮点

- 悬浮聊天窗（微信风 UI）
- 三种调用方式
  - 复用主 API
  - 主 API 直连（不吃主预设）
  - 独立 API
- Prompt 规则预设
  - 内置预设模板（可一键应用）
  - 支持保存自定义预设
- 输入过滤
  - 可移除 `<snow>...</snow>` 等标签内容
  - 支持自定义标签列表（英文逗号分隔）
- 上下文增强
  - 可读取最近剧情楼层
  - 注入吐槽历史记忆
- 历史持久化
  - 本地文件（`rant-history.json`）优先
  - 浏览器存储兜底
- 移动端适配（小屏布局优化）

## 安装方式

将以下文件放到扩展目录：

- `manifest.json`
- `rant.js`
- `rant.css`

目录示例（Windows）：

`E:\SillyTavern\SillyTavern-release\data\default-user\extensions\chajian`

然后重启或刷新 SillyTavern 页面。

## 基础使用

1. 点击悬浮窗右上角 `⚙` 打开设置
2. 选择调用模式并配置 API（如需）
3. 在 `AI回复预设` 里选择模板并应用，或自行编辑规则
4. 输入想吐槽的剧情，点击 `发送`

## 手机使用说明

- 已内置小屏适配（`max-width: 768px`）
- 手机上会禁用桌面拖拽/缩小交互，优先保证输入与阅读体验
- 设置弹窗与预览弹窗自动扩大为接近全屏

## 过滤标签注意事项

自定义标签必须使用**英文逗号**分隔，例如：

`snow,details,ooc`

如果使用中文逗号 `，`，可能导致过滤不生效。

## 数据保存说明

- 吐槽历史文件：`rant-history.json`
- 设置与兜底历史：`localStorage`

如果运行环境无法写文件，会自动回退到浏览器本地存储。

## 常见问题

### 1) 为什么会混入主剧情风格？

如果使用“复用主 API”，模型可能受主预设影响。  
建议改用“主 API 直连（不吃主预设）”或“独立 API”。

### 2) 为什么过滤标签没生效？

请检查：

- 是否勾选“发送前移除标签”
- 标签列表是否用英文逗号
- 标签是否成对闭合（如 `<snow>...</snow>`）

### 3) 聊天历史存在哪？

优先写入插件目录 `rant-history.json`，同时保留浏览器存储兜底。
